> **提示：**格点天气的空间分辨率为3-5公里。

> **注意：**格点天气是一种数值预报，由卫星、气象雷达等数据通过模型算法运算而来，它可能受到许多因素的影响，如山脉、建筑物、人类活动、大气变化等。格点天气与观测站数据会有所不同，不可直接对比。
{:.bqwarning}
