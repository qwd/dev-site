获取最近10天的中国空气质量历史再分析数据。

> 例如今天是12月30日，最多可获取12月20日至12月29日的空气质量历史数据。


| 接口代码 | 接口                  | 数据类            |
| ---------------- | ---------------- | ----------------- |
| historicalAir| 历史空气质量数据  | HistoricalAirResponse |
