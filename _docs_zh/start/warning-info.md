不同国家的灾害预警具有不同定义的等级和类型，这里可以了解到全部灾害预警信息的等级和类型说明。

> 请注意，所有预警类型和等级都有可能发生变化，包括新增、修改或删除，因此请务必调整好你的程序，以便在发生变化的时候不会导致错误出现！
{:.bqwarning}

> 并非所有预警的最低或最高等级是蓝色或红色，例如霾预警只有黄色和橙色等级

## 预警等级 {#warning-level}

| Level | 适用区域（iso3166） |
| ----- | ------------------- |
| 白色  | CN, 广东省          |
| 蓝色  | CN                  |
| 绿色  | CN, EU, RU          |
| 黄色  | CN, EU, RU          |
| 橙色  | CN, EU, RU          |
| 红色  | CN, EU, RU          |
