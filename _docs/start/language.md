和风天气支持中英德法西意日韩俄泰等30多种语言，以下是在API或SDK中的多语言参数。

> 默认语言为中文，如一些城市名称没有中文的，使用英文或本地语言代替。
> 
> 某些数据（如生活指数、灾害预警的详情）中的部分内容仅支持本地语言或英语。

| 简体中文  | zh-hans、zh  |LANGUAGE_TYPE_ZH     |ZH_HANS  |
| 繁体中文  | zh-hant       |LANGUAGE_TYPE_ZHHANT |ZH_HANT  |
