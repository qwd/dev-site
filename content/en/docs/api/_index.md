QWeather web API is a simple RESTful API to access location-based weather data, including geolocation, real-time weather, weather forecasts, air quality, severe weather warnings, minute precipitation, weather index and other weather data services.



*See [Deprecated](/en/docs/deprecated/) for deprecated API/SDK.*
